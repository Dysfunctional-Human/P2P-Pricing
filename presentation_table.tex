%% KEY OBSERVATIONS
\section*{Key Observations}

\begin{enumerate}
\item \textbf{SDR consistently achieves highest raw savings} (4.8--23.5\% in net-importing communities), but fairness degrades in unbalanced compositions.
\item \textbf{MMR provides the best fairness--savings balance} --- recommended in most scenarios when fairness is weighted. Consumer and prosumer savings stay within 1--2\% of each other.
\item \textbf{Bill-Sharing systematically disadvantages prosumers} (only 3.2\% savings vs 8.5\% for consumers), yielding poor fairness (0.547) in mixed communities.
\item \textbf{DSM effect saturates below $\alpha \approx 1$} --- further reducing $\alpha$ yields no additional load shifting. Above $\alpha = 5$, DSM progressively disengages.
\item \textbf{Wider grid tariff spread benefits all P2P mechanisms} --- the buy--sell gap defines the maximum value that P2P trading can capture.
\item \textbf{Consumer-heavy communities see lower absolute savings} (4--6\%) because there is less local generation to trade.
\end{enumerate}
